Mathematical Models

  • Fisheries Food webs: Modeling aquatic communities to study broader effects of fishing and harvesting:
    • Bio-Economics Model: Model written in Matlab. Code available at github link.
  • Plant-Pollinator Networks: Network models to investigate the dynamics of large communities of interacting plants and pollinators:
    • Phenology Model: Model written in Matlab. Code available at github link.
  • Plant Ontogeny: The models I have used to study the effects of plant ontogeny on plant demographics and herbivory dynamics:
    • Multiple Herbivores: Dryad Link. The files provided here are Wolfram .cdf files (computable document format). A CDF is a free public format that can be used with Wolfram's free CDF Player, available at https://www.wolfram.com/player/.
